## Build
>To build it you have to use CMake, or Bash.
(execute it from **$project folder$/bin** )


#### **Windows:**
```
cmake ../ && cmake --build ./ && start ./Debug/TME.exe
```
---
#### **UNIX-like:**
```
cmake ../ && cmake --build ./ && ./TME
```

There are more information in instruction.pdf or example.txt .
