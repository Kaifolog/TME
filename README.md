TME - Turing machine emulator.

To build it you have to use CMake, or Bash.

(execute it from project folder)


Get to /bin folder, then:

Windows:

cmake ../
cd ../
cmake ./bin && cmake --build ./bin && start ./bin/Debug/main.exe
